# Kubernetes Learning Resources: Zero to Advanced (Free, March 2026)

A comprehensive, verified guide to learning Kubernetes for free — from absolute beginner to advanced level.

---

## Recommended Learning Path

| Phase | Weeks | Time | Resources |
|---|---|---|---|
| Foundation | 1–2 | 10–15h | Official tutorials + freeCodeCamp video |
| Core concepts | 3–5 | 20–25h | LFS158x + Civo Academy + Killercoda |
| Intermediate | 6–8 | 25–30h | KubeAcademy + freeCodeCamp Handbook + iximiuz |
| Advanced | 9–16 | 40–50h | Helsinki MOOC + DevOpsCube + GitHub paths |
| Cert prep | 17+ | 40–100h | KCNA → CKA/CKAD |

**Total:** ~150–200 hours from zero to job-ready/cert-ready.

## Phase 1 — Foundation (Weeks 1–2)

### 1. Kubernetes Official Tutorials
- **URL:** https://kubernetes.io/docs/tutorials/kubernetes-basics/
- **Level:** Beginner
- **Format:** Interactive browser-based labs (no install needed)
- **Duration:** 5–10 hours
- **Cost:** Free
- **What you learn:** Deploying containerized apps, scaling, updating, debugging
- **Why start here:** Directly from Kubernetes maintainers; real embedded terminal

### 2. Linux Foundation — Introduction to Kubernetes (LFS158x)
- **URL:** https://www.edx.org/learn/kubernetes/the-linux-foundation-introduction-to-kubernetes
- **Level:** Beginner
- **Format:** Self-paced video course + hands-on labs
- **Duration:** 28–42 hours (14 weeks, ~2–3h/week)
- **Cost:** Free to audit (certificate upgrade: $189)
- **Enrollment:** 410,000+ students | Rating: 4.4/5
- **What you learn:** Architecture, components, Minikube setup, CNCF ecosystem

### 3. freeCodeCamp — Kubernetes for Absolute Beginners (Video)
- **URL:** https://www.freecodecamp.org/news/learn-docker-and-kubernetes-hands-on-course/
- **Level:** Beginner
- **Format:** Live-coding video
- **Duration:** 6 hours
- **Cost:** Free (YouTube/freeCodeCamp)
- **Instructor:** Guy Barrette (25+ years experience)
- **What you learn:** Docker fundamentals + complete Kubernetes introduction

## Phase 2 — Core Concepts (Weeks 3–5)

### 4. Civo Academy
- **URL:** https://www.civo.com/academy
- **Level:** Beginner–Intermediate
- **Format:** 50+ free video guides
- **Duration:** 20–30 hours
- **Cost:** Free
- **Instructors:** Saiyam Pathak, Kunal Kushwaha
- **Modules:**
  - Before Kubernetes (Linux and Docker basics)
  - Kubernetes Introduction (CNCF landscape)
  - Kubernetes Concepts (architecture, objects, namespaces, labels)
  - Kubernetes Setup (Minikube, kubeadm, containerd, kubectl)
  - Kubernetes Configuration and Security (ConfigMaps, Secrets, RBAC)

### 5. freeCodeCamp — The Kubernetes Handbook
- **URL:** https://www.freecodecamp.org/news/the-kubernetes-handbook/
- **Level:** Beginner–Intermediate
- **Format:** Written handbook + embedded video tutorials
- **Duration:** 6–8 hours
- **Cost:** Free
- **What you learn:** Pods, Services, Deployments, PersistentVolumes, Ingress, kubectl, YAML
- **Project:** Real-world Express.js + PostgreSQL multi-service deployment

### 6. KubeAcademy (VMware/Broadcom)
- **URL:** https://kube.academy/paths
- **Level:** Beginner–Advanced
- **Format:** Video lessons + hands-on labs + progress tracking
- **Duration:** 16–24 hours
- **Cost:** Free
- **Paths available:**
  - Understanding Cloud Native
  - Kubernetes Core Concepts

## Phase 5 — Certification Prep (Optional, Weeks 17+)

> All exams include 1 free retake. kubernetes.io docs are allowed during the exam.

### KCNA — Kubernetes & Cloud Native Associate
- **Exam cost:** $250
- **Prep time:** 4–8 weeks
- **Passing score:** 75%
- **Domains:** Kubernetes Fundamentals (46%), Container Orchestration (22%), Cloud Native Architecture (16%), Observability (8%), Application Delivery (8%)
- **Free prep:** CNCF curriculum on GitHub, KodeKloud free, freeCodeCamp
- **Coursera free audit:** https://www.coursera.org/learn/pearson-kubernetes-and-cloud-native-associate-kcna-complete-course

### CKA — Certified Kubernetes Administrator
- **Exam cost:** $445
- **Prep time:** 8–12 weeks
- **Format:** Performance-based, 2 hours, 15–20 hands-on tasks in remote shell
- **Free prep:** TechiesCamp CKA guide, DevOpsCube CKA study guide, GitHub repos above

### CKAD — Certified Kubernetes Application Developer
- **Exam cost:** $395
- **Prep time:** 6–10 weeks
- **Free prep:** CKAD-exercises repo, KodeKloud, freeCodeCamp
